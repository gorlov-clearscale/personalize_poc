# Personalize POC

## Install

Depending on your OS, run:
```shell script
./install_macOS.sh
# OR
./install_ubuntu.sh
```

## Deploy

```shell script
./deploy.sh
```
After training the models in the deployed Notebook, proceed to `./EndpointsKit/README.md` and follow its instructions
